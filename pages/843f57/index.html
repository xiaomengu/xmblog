# 图片和文字水平居中对齐

通过flex布局实现图片与文字水平对齐

只需要在父级元素中css添加：

display:flex;

flex-direction:row; /* 排列方向水平 */

align-items:center;/* 垂直居中 */

# 元素水平垂直居中

只需要在父级元素中css添加：

display:flex;

justify-content: space-between; /* 两端居中 */

justify-content: center; /* 水平居中 */

            :class="$route.name === item.name ? 'bread' : 'bread-normal'"
            v-for="item in tabList"
            :key="item.path"
            :to="{ path: item.path }">

# ElementUI date-picker报错

控制台报错：
Avoid mutating a prop directly since the value will be overwritten whenever the parent component re-renders. Instead, use a data or computed property based on the prop's value. Prop being mutated: "placement"
1
2

原因：ElementUI版本问题，降到 2.14.1就好了

npm install element-ui@2.14.1 -s
1
